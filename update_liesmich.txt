Update:	01ACP V 1.2.1 nach V 1.3.0
==================================

!!! ACHTUNG !!!
Bei Einem Update des 01ACP auf die Version 1.3.0 MÜSSEN gleichzeitig ALLE verwendeten Module ebenfalls
auf die neuste Version aktualisiert werden!
Bereiten Sie zuerst alle Update-Dateien vor, bevor Sie mit dem Update beginnen!
Die aktuellste Version der Module kann direkt von http://www.01-scripts.de heruntergeladen werden.
!!! ACHTUNG !!!

- Legen Sie ein Backup ihrer Datenbank an!
- Legen Sie ein Backup ihrer Daten via FTP-Programm an!

- Löschen Sie das Verzeichnis 01scripts/01acp/system/tiny_mce/*

- Markieren Sie alle im Verzeichnis 01scripts/ enthaltenen Unterverzeichnisse
!!NICHT die Datei 01scripts/01_config.php kopieren! Für ein Update die bereits existierende Datei behalten!!

- Laden Sie die markierten Verzeichnisse per FTP-Programm auf Ihren Webspace hoch. Überschreiben Sie dabei
  alle bereits existierende Daten (AUßER die Datei 01_config.php !).

- Rufen Sie die Datei 01scripts/01acp/update.php
  in Ihrem Browser auf und folgen Sie den weiteren Anweisungen

- Löschen Sie nach erfolgreichem Update folgende Dateien & Verzeichnisse mit Ihrem FTP-Programm:
  * 01scripts/01acp/_up_data.php

- Loggen Sie sich danach in den Administrationsbereich ein und aktualisieren Sie die von ihnen verwendeten Module!
  Erst wenn Sie die Module aktualisiert haben, können Sie sie wieder wie gewohnt nutzen!
